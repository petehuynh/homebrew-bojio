# homebrew-bojio

Homebrew tap for Bojio desktop apps.

## Install

```bash
brew tap petehuynh/bojio
brew install --cask aether
```

## Casks

| Cask | Description |
|------|-------------|
| `aether` | Household AI — run open language models on your Mac and share over Wi-Fi |
| `bojio-edge` | Distributed LLM inference node for the Aetheria network |

## Uninstall

```bash
brew uninstall --cask aether
brew uninstall --cask bojio-edge
```
